## 题材核心

心理小说写的是**意识与记忆的内部地形**：一个人如何被自己的过去塑形、误读自己的当下、试图修补又再次撕裂。它的张力不在外部事件，而在认知与现实的距离。

## 主题维度

- 主线张力候选：创伤后的重建与复发 / 记忆的不可靠 / 自我认同的分裂 / 抑郁/焦虑/解离作为一种存在方式 / 治疗关系中的权力 / 家庭代际的心理传递
- 必须避免：把心理学诊断词当主题（不要写"抑郁症患者的故事"，写"一个具体的人在某段时间里的具体处境"）
- 主题通过节奏的失常、感知的扭曲、不肯说出口的事被传递

## 题材禁忌

- 病理标签化：禁止给主角贴"PTSD/抑郁症/焦虑症"的标签作为身份
- 治疗师万能：不要让心理治疗师成为说教者或解谜者
- 创伤戏剧化：童年创伤的揭示不要安排成"高潮反转"
- 内心独白填空：不要让心理活动变成"分析报告"——心理小说最大的敌人是评论性语言

## 心理真实

- 创伤记忆是碎片的、突发的、由感官触发的（一种气味、一种声音、一种光线）
- 同一事件在不同时间被想起，细节会改变；这是真实的而不是错误的
- 解离不是失去意识，而是身体在场而自我"被推到旁边"
- 抑郁不是悲伤，而是世界变薄、欲望失重
- 焦虑不是害怕某件事，而是身体先反应、大脑后追问"为什么"

## 语言铁律

- 内心独白绝不能写成分析。✗"她意识到这是童年的投射" → ✓"她又闻到了那种煤油味，胃像被人轻轻按了一下"
- 用感官代替结论。让读者自己拼出心理学
- 不要在叙事中诊断角色；不要让其他角色为主角"贴标签"
- 时间错位用句法表达——长句可以横跨年代，短句可以钉在此刻
- 沉默和"未说出的话"比说出的话承担更多

## 叙事指导

允许非线性结构：闪回、跳跃、重复、回旋。但跳跃必须有一个具体的外部锚点（一个动作、一个气味、一句话）作为引信。

让读者短暂迷路是好事，但不能持续迷路——每一段意识流都应该在结束前让读者重新触到地板。

意象网络比情节更重要。一个井、一面镜子、一种声音可以在全书反复出现，每次承担略有不同的心理位置。

人物之间的对话常常是错位的：一个人说 A，另一个人接 B，他们都没听见对方真正说的是什么。这种错位是心理小说的常态。

结局通常不是"治愈"，而是"理解到自己将一直与之共处"。圆满的心理叙事是可疑的；好的心理小说承认伤口的开放性。
